# 🦇 BatMan Originals Website

A cinematic, story-driven multi-page website  built using HTML, CSS, and Bootstrap inspired by the concept of **Batman as a symbol, not just a hero**.

This project represents a modern, responsive, and visually engaging web experience built using **HTML, CSS, and Bootstrap**, designed to showcase both UI skills and creative storytelling.

---

## 🌐 Live Demo
👉 https://priyanshung1480.github.io/batman-originals-website/

---

## 🚀 Features

- 🎨 Cinematic Batman-themed UI design
- 📱 Fully responsive (mobile-friendly)
- 🧭 Multi-page navigation system
- ✨ Smooth scroll animations
- 🦇 Custom Bat-Signal glowing effect
- 📦 Reusable components (Navbar, Cards, Footer)
- 🧑‍💻 Clean and structured code

---

## 📄 Pages Included

- Home Page (Hero + Story + CTA)
- Services Page (Training & Capabilities)
- About Us Page (Concept & Vision)
- Contact Page (Form + Info)
- Registration Page (Modal-based form)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Bootstrap 5
